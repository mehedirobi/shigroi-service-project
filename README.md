#  Shigroi Service

A full-stack service marketplace platform where users can quickly find and hire local service providers such as cleaners, carpenters, electricians, and masons. Users can also register as service providers and offer their services.

This platform connects service seekers and service providers in a fast, simple, and scalable system.

---

## ✨ Key Features

### 👤 User Features
- Browse and search available services
- Book services with simple workflow
- View service provider profiles
- Authentication system (login/register)

### 🧑‍🔧 Service Provider Features
- Register as service provider
- Add and manage services
- Receive service requests

### ⚙️ System Features
- Role-based access (User / Provider)
- Secure authentication system (JWT)
- REST API integration
- Responsive UI for all devices
